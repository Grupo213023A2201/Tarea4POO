# Sistema de Gestión de Servicios

## Programación Orientada a Objetos – Fase 4

Este proyecto implementa un sistema de gestión de reservas para:

- Salas
- Equipos
- Asesorías

Aplicando los conceptos de:

- Clases abstractas
- Herencia
- Polimorfismo
- Excepciones personalizadas
- Encadenamiento de excepciones (raise from)
- Registro de eventos en logs
- Manejo de errores con try / except / else / finally

## Estructura del sistema

El sistema permite:

1. Registrar clientes con validación de datos
2. Crear servicios con cálculo de costos
3. Realizar reservas
4. Controlar errores mediante excepciones
5. Registrar eventos en archivo logs.txt
